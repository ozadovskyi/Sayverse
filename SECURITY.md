# Security policy

OpenTranslator is bring-your-own-key — the OpenAI API key is entered on-device
and never leaves it. There is no backend that handles secrets.

## Reporting a vulnerability

If you find a security issue — anything that could expose a user's API key,
intercept user audio, or allow execution of untrusted code on a user's
device — please report it privately rather than opening a public issue.

Use GitHub's [private vulnerability reporting](https://github.com/ozadovskyi/OpenTranslator/security/advisories/new)
on this repository. Expect an acknowledgement within a week.
